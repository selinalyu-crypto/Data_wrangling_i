Rewrite, combine, and extend (add mutate)

``` r
pulse_df = 
  haven::read_sas("./data/public_pulse_data.sas7bdat") |>
  janitor::clean_names() |>
  pivot_longer(
    bdi_score_bl:bdi_score_12m,
    names_to = "visit", 
    names_prefix = "bdi_score_",
    values_to = "bdi") |>
  relocate(id, visit) |>
  mutate(
    visit = replace(visit, visit == "bl", "00m"),
    visit = factor(visit)) 
```
